# Installation Notes
First install nodejs the normal way for your operating system.

## OS X with Homebrew

```
$ brew install node
```

## Ubuntu Linux

```
$ sudo apt-get update
$sudo apt-get install nodejs
```

##Install the bot and dependencies

```
$ cd path/to/standupbot
$ npm install
```
